This directory contains data files analyzed in:

Barba-Montoya et al. (2024). Integrating phylogenies with chronology to assemble the Tree of Life. https://doi.org/...

Four directories for simulated data:

  timeree_collection_c2 -> six MSAs, six constituent timetrees, and one model supertree.
  timetree_collection_c1 -> six MSAs, six constituent timetrees, and one model supertree.
  timetree_collection_c3 -> six MSAs, six constituent timetrees, and one model supertree.
  toy_model_timetree_collection -> five constituent timetrees, and one toy model supertree.
